Color Quantization

Color Quantization is the process of reducing number of colors in an image. One way to accomplish this is K-means clustering, which can separate colored data into groups of distinct colors. This can be pretty useful for color blind people (such as me), who have trouble distinguishing shades of red from shades of green, failing to see any number in pictures like the one on the left side of the animated image. These pictures are used as tests to diagnose color blindness.

A simple web app is created for color quantization using only up to 5 basic colors in an attempt to facilitate the detection of numbers in 20 different color blind tests.
